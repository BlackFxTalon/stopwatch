Stopwatch - небольшое приложение - таймер с тремя кнопками Start (начало запуска таймера), stop(остановка таймера), reset (cброс времени у таймера), которое я написал в ходе изучения темы асинхронного программирования на MDN. Приложенька с голым UI дизайном, для теста. Используются js функции  - setTimeOut(), setInterval(), clearTimeOut(), clearTimeInterval(). Ссылка на netlify-server проекта для теста - .
